# Instructions for Adding to Config

If your prompt files have different names, update `prompts/config.yaml`:

```yaml
prompts:
  paraphrase: your_paraphrase_name.txt
  language_fix: your_language_fix_name.txt
  hallucination_check: your_hallucination_name.txt
```
