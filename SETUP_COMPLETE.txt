🚀 QUICK START (3 STEPS)
════════════════════════

1. Install dependencies:
   pip install -r requirements.txt

2. Set API key (if not already set):
   export ANTHROPIC_API_KEY="sk-ant-your-key"

3. Run on your data:
   python qa_cleaner.py your_data.csv -o output.csv

🎯 HOW PROMPT SYSTEM WORKS
═══════════════════════════

┌─────────────────────────────────────┐
│   prompts/config.yaml               │
│   (Define which prompts to use)     │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│   PromptManager (prompt_manager.py) │
│   (Loads prompts from files)        │
└─────────────────────────────────────┘
           ↓
┌─────────────────────────────────────┐
│   qa_cleaner.py                     │
│   (Uses prompts for LLM calls)      │
└─────────────────────────────────────┘

Result: Easy to swap/customize prompts without touching code!

💡 COMMON TASKS
═══════════════

USE COLLEAGUE'S PROMPTS:
  cp ~/downloads/*.txt prompts/
  python qa_cleaner.py data.csv -o output.csv

TRY ALTERNATIVE PROMPT:
  cp prompts/examples/paraphrase_technical.txt prompts/paraphrase.txt
  python qa_cleaner.py data.csv -o output.csv

CUSTOMIZE A PROMPT:
  # Edit prompts/paraphrase.txt in your editor
  python qa_cleaner.py data.csv -o output.csv

BACKUP & RESTORE:
  cp -r prompts prompts_backup
  # ... make changes ...
  cp -r prompts_backup prompts

🔑 KEY CONCEPTS
═════════════

ACTIVE PROMPTS (prompts/*.txt)
  → These are used by the script
  → Edit/replace these to change behavior
  → Names/paths referenced in config.yaml

EXAMPLE PROMPTS (prompts/examples/*.txt)
  → Reference alternatives
  → Copy into prompts/ to use them
  → No automatic changes

CONFIG FILE (prompts/config.yaml)
  → Maps prompt names to files
  → Central control point
  → Easy to switch between versions
